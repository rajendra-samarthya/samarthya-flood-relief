# Samarthya — Marathwada Flood Relief 2026

This repository hosts the interactive impact dashboard for Samarthya's flood relief response in Osmanabad (Dharashiv), Maharashtra, India.

## About

**Organisation:** Samarthya — Women-led, youth-initiated grassroots nonprofit  
**Programme:** Madat (Relief & Rebuilding)  
**Operation:** Marathwada Emergency Flood Relief 2025–26  
**Coverage:** 3,000 households · 55 habitations · Omerga & Lohara blocks, Osmanabad

## Dashboard

The dashboard covers:
- Overview & key statistics
- Reach & vulnerability profiling
- Geographic map (GPS-tagged distribution points)
- Samarthya team profiles
- Accountability & evidence
- CHS compliance documentation
- Context & field gallery

## Communities served

De-notified and Nomadic Tribe (DNT) communities including Laman Banjara, Wadar, Berad, and Mariaaiwale — historically excluded from mainstream relief systems.
